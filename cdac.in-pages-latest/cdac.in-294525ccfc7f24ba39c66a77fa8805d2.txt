Abhayam
Help in Distress
ABHAYAM:  It is a software system to assist anyone in distress by sending alert/SOS to pre-defined contacts/Police via SMS and GPS.
Voice activated app
ABHAYAM - VICTIM is a client side mobile application which is used to track user while in trouble.
Multiple Rescuers:  This option has rescuer (s) information with the provision of adding and deleting.
Configurable SMS Lag time and password protected Secured System  This application will send the tracking information to the rescuers (and server) at customized interval
Once the application is activated, three main buttons are visible namely:
SOS:  To track the user and keep on sending SMS to rescuers including Server.
Track Me:  To track the user and keep on sending SMS to rescuers (Excluding server).
Exit:  This is used to exit application.
If user clicks exit button when rescue is going on, the application will work as backend service until victim confirms "Rescue done".
If user confirms rescue completion then ABHAYAM application will stop sending SMS and send a confirmation message (normal SMS) to all the rescuers.
Other menu options:  For user convenience, a few more options are available:
Camera:  This option will open camera along with preview for taking location as a proof
Quick E-eye:  This will capture and save the picture immediately without preview mode
Voice Recording:  To record voice and save in local folder automatically
Call Rescuer: Make phone call automatically to available rescuer.
ABHAYAM- RESCUER  is a separate mobile application which can track victims by rescuers/server.
If ABHAYAM message is received, it will open the ABHAYAM -Rescuer application window along with alert sound, location information on map.
On each message, it updates the valid victim (s) with latest location.
Standard navigation tools are available.
This option provides available victims list.
On selecting a victim, it will zoom to the selected victim location.
Save and Open Track:  This feature is used to save and open victim (s) track into local drive for future use.
