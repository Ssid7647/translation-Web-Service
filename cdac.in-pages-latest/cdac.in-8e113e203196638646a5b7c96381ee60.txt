Five days workshop on Big Data Technologies and Machine Learning - 2019
Five days workshop on “Big Data Technologies and Machine Learning - 2019”
Date: 02 - 06 December 2019
Workshop Objective
The workshop has been specifically designed for the industry, academia, scientific and research communities for understanding of Big Data concepts and technologies, its working and advances, Machine Learning and Deep Learning.
The detailed five-day program focuses on practical knowledge of Big Data and Machine Learning tools such as Apache Hadoop, Apache Spark, Apache Hive, Scikit-learn, SparkML and Keras.
The program has been designed to delve deeper into Machine Learning algorithms with real-world use cases, along with hands-on sessions and Deep Learning concepts.
Target Audiences
Scientists/ Researchers/ Academicians/ Tech-enthusiasts keen to learn and adopt Big Data tools, Machine Learning and Deep Learning
Pre-requisites
Familiarity with Linux Platform
Working knowledge of Python
Basic knowledge of Java
Intake
Participants will be limited to a maximum of thirty (30) seats and on First-come First-served basis only.
** Theory sessions will be followed by hands-on session
