Workshop on "Smart Grid: Applications & Cyber Security"
Date: March 17th , 2017 @ C-DAC Knowledge Park, Byappanahalli, Old Madras Road, Bangalore
C-DAC Bangalore, is conducting a one-day workshop on " Smart Grid: Applications & Cyber Security " with the main focus on latest applications in smart grid and cyber security
Course Contents:
This program enables participants to get the latest research and trends towards smart grid in Indian power sector .The main topics include:
Self-healing Grid
M2M Communications in Power Sector
Smart Grid Standards & Protocols
Role of Smart Grids in Distribution
Smart grid cyber security
Data Science for Smart Grid
Grid of Things (GoT)
Research Trends in Wide Area Monitoring Systems (WAMS)
Target Audience:
The program is designed for Practitioner, Academics and Researchers
