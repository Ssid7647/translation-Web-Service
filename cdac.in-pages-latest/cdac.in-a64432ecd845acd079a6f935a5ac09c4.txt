Sample codes
Sample test codes available with toolkit helps developers to try on some existing code.
This greatly helps in understanding the building blocks of toolkit using which applications can be built.
Guidelines given along with the test codes explain the purpose of components in the test codes.
This enables developer to quickly identify important components, sequence of steps to be taken while programming using toolkit.
