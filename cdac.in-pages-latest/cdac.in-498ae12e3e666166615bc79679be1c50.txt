GIST, JAVA APPLET FOR TYPING IN INDIAN LANGUAGES ON THE WEB
This is a set of light weight JAVA Applets for web application development.
It allows the creation of interactive applications for the Internet in Indian languages in various environments.
It supports INSCRIPT keyboard layout and the Phonetic English (like a word is pronounced in English).
You can use JAVA applet on your Indian language Website / web application for getting input from a variety of users - quickly.
Applet downloads fast and allows user to type Indian languages (Hindi, Marathi, Gujarati, Assamese, Bengali, Kannada, Tamil, Telugu, Oriya, Malayalam, Punjabi, etc.) using Phonetic English or INSCRIPT keyboard layout.
It is ideal for feedback e-mail forms, chat applications, etc. in Indian Languages.
It will work on all Java Enabled browsers irrespective of the server side technology used (ASP, JSP, PHP, .Net, etc.). It is compatible with all GIST Technology based server side conversion, transliteration routines.
The same components will be used across all browsers and OS.
User can type Indian languages in various OS (such as Windows - 95/98, Win2K, XP and Linux - RedHat, Fedora, etc.) and browsers (such as IE, Netscape, Mozilla, FireFox, etc.).
