Language Lab for Rajbhasha
Language lab for Rajbhasha is a language learning solution with specially equipped facility designed to train students in Hindi language using proven language teaching techniques.
It is an audio-visual installation used as an aid in modern language teaching.
The Language Laboratories for Rajbhasha Vibhag supports the development and implementation of tools and materials necessary for the expansion of effective distance / distributed education by Central Hindi Training Institute.
Language Lab for Rajbhasha is a LAN based language lab mainly having three major modules.
These three main modules are again sub divided into various sub modules equipped with lots of functionalities.
These sub modules allows the both way communication.
Admin console
Teacher console
Student console
Salient Features:
Allow teacher to create different groups.
Both way communication
Different modes of communications like Audio message, Video message and Text message.
Raise hand facility for student
Lesson creation
Student monitoring
Student evaluation
Live video Lecture over LAN
Text lessons
Video Lessons
Audio Lessons
Various exercises and Test
Transfer of files between students and teachers
