GIST Text To Image Generator
This utility converts the input Indian language data to an image.
The advantage of this utility is one can create an image of the Indian Language data and can send it as an email for the client to read or send it to any system or device capable of showing the image.
And being an image, no font is required to be present on the receiving system.
